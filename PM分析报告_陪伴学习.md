## 1. 产品全景

### 1.1 一句话定位

**陪伴学习**（WhiteNoise）是一款沉浸式白噪音 + 视觉场景辅助专注工具，帮助用户在自习、工作、放松、入睡时通过多感官环境音画提升专注力。

### 1.2 目标用户群

| 用户画像 | 场景 | 核心需求 |
|----------|------|----------|
| 在校学生（大学/考研/考公） | 自习室、图书馆、宿舍 | 隔离环境噪音，营造沉浸学习氛围 |
| 远程工作者 / 程序员 | 居家办公、开放式工位 | 快速进入心流状态，缓解焦虑 |
| 冥想 / 放松用户 | 睡前、休息间隙 | 平静情绪，辅助入眠 |
| 白噪音爱好者 | 日常背景音 | 自由混音，丰富听觉体验 |

### 1.3 核心场景与功能清单

| # | 核心场景 | 功能覆盖 |
|---|----------|----------|
| 1 | **沉浸场景浏览** | 7 个视觉场景（壁炉·冬夜 / 山涧溪流 / 海浪沙滩 / 雪地夕阳 / 森林雪夜 / 角落咖啡 / 雨中咖啡），视频 + 粒子双引擎渲染，支持切换、跨渐过渡、手势滑动 |
| 2 | **白噪音自由混音** | 100+ 音效源（雨声 / 自然 / 动物 / 城市 / 场所 / 交通 / 物品 / 噪声 / 双耳节拍），每场景独立混音配置，开关 + 音量滑块 |
| 3 | **计时专注工具** | 4 种模式（时钟 / 正计时 / 倒计时 / 番茄钟），开始/暂停/重置，倒计时震动提醒 |
| 4 | **聚焦模式** | 全屏大字时钟（2 种样式），屏幕渐暗（3 级），屏蔽视觉干扰 |
| 5 | **多端适配** | Web PWA / iOS / Android（Capacitor），横屏锁定，移动端粒子降级 |

### 1.4 技术栈 / 平台

| 层级 | 技术选型 |
|------|----------|
| **移动端** (StudyCompanion) | Expo (React Native) + expo-av + expo-screen-orientation + expo-haptics + AsyncStorage |
| **Web 次世代** (white-noise) | Vue 3 + TypeScript + Vite + Pinia + Three.js + ShaderMaterial |
| **Web 原始版** (web/) | 纯 Vanilla JS + Canvas 2D + CSS 变量 |
| **后端** | Express 静态服务 + 腾讯云 COS 预签名 URL 代理/重定向 |
| **构建 / 部署** | Vite build / Expo publish / Capacitor native sync |
| **云存储** | 腾讯云 COS（视频/音频素材托管） |

## 2. 用户旅程（3 条核心路径）

### 路径 1：学生自习专注

```
打开应用 → 选择场景 → 调节声音 → 开始计时 → 进入专注状态
    ↓           ↓           ↓           ↓            ↓
   😊          😊          😐          😊           😊
[启动快]   [场景丰富]  [混音面板层级深] [操作直觉] [自动隐藏控制栏]
```

- **总体满意度**：7/10
- **最痛的点**：调节声音需打开底部面板，每切换一个音效都要点击开关并拖曳滑块，操作成本高。
- **建议修复方案**：在 CtrlBar 上增加一级快捷音量控制（滑动条直接调当前场景总音量），将常用音效（当前场景预设）默认展示在面板首屏。

### 路径 2：睡前放松

```
打开应用 → 切换到海景 → 静音视频 → 开启倒计时 → 调暗屏幕 → 闭眼聆听
    ↓           ↓           ↓           ↓            ↓           ↓
   😊          😊          😊          😐           😊          😊
[快速加载] [视觉效果沉浸] [一键静音] [倒计时预设] [3级渐暗] [声音持续播放]
```

- **总体满意度**：8/10
- **最痛的点**：倒计时结束后屏幕不会自动熄灭，用户需手动锁屏。番茄钟完成无持久化统计。
- **建议修复方案**：增加"闭眼模式"——倒计时结束后自动降低亮度至 0 并显示纯黑屏。番茄钟历史数据存入 localStorage，显示今日专注总时长。

### 路径 3：快速进入专注（极简路径）

```
打开应用 → 点击聚焦按钮 → 屏幕变暗 → 埋头工作
    ↓           ↓            ↓           ↓
   😊          😊           😊          😊
[秒开]    [一键切换]    [自动工作]   [无干扰]
```

- **总体满意度**：9/10
- **最痛的点**：聚焦模式下如需切换场景或调整声音，必须先退出聚焦模式，操作后需重新进入。
- **建议修复方案**：聚焦模式下允许通过快捷手势（双指点击 / 长按）临时调出音量控制，操作完后自动恢复聚焦。

### 各模块详情

#### 3.1 场景渲染

- **可用性**：正常工作。7 个场景通过 video → canvas 绘制实现视频背景；white-noise 版使用 Three.js 粒子系统做篝火场景的 Emitter 渲染。
- **体验**：视频画质高、色彩氛围好。粒子系统篝火效果目前有 debug 橙色球残留，火焰形状尚未达预期（粒子无锥形蓬松形态）。
- **完整性**：缺少每个场景的粒子版本（目前仅 campfire 实现了 Emitter 架构；ocean / rainy-window / forest 有配置定义但 particleSampler 返回空数组，实际未运行）。
- **一致性**：视频版所有场景风格统一；粒子版仅 campfire 可用，其他场景的粒子渲染路径未完成。
- **性能**：视频版首次加载需缓冲（依赖 COS 预签名 URL），约 1-3 秒。粒子版 Three.js 打包体积约 628KB，移动端有 `isMobile` 降级判断。

#### 3.2 声音混音

- **可用性**：正常工作。100+ 音效覆盖雨声/自然/动物/城市/场所/交通/物品/噪声/双耳节拍 9 大类，每场景 8 个预设音效自动激活。
- **体验**：底部面板从底部滑出（cubic-bezier 缓动），有 grip 手柄 + 关闭按钮。网格布局 80px 最小宽度，响应式适配。每个音效有 icon + label + 音量滑块。
- **完整性**：mix 版本号控制，localStorage 持久化。音效路径回退到 cat 目录，支持 .mp3 / .wav。
- **一致性**：所有场景共用同样的 SoundPanel 组件，交互一致。

#### 3.3 计时工具

- **可用性**：4 种模式全部可用。时钟显示实时时间；正计时有秒级精度；倒计时可设时分；番茄钟默认 25 分钟，完成后触发 haptics + vibration（Expo 版）。
- **体验**：56px 大字体，tabular-nums 等宽数字，视觉美观。模式切换有标签页样式。
- **完整性**：倒计时/番茄钟的时间设置 UI 存在但非响应式（`v-model` 绑定缺失，输入值未实际驱动 `timer.duration`）；番茄钟统计功能有 `timerStats` 容器但内容为空。

#### 3.4 聚焦模式

- **可用性**：完美。点击底部聚焦按钮轮换三种状态（关闭 / 卡片 / 大卡片），全屏聚焦时钟显示当前时间，每 10 秒更新一次。
- **体验**：backdrop-filter 毛玻璃效果，88-112px 超大字号，平滑 scale 动画。点击聚焦卡片可切换样式。
- **完整性**：聚焦模式下无其他控制元素干扰，纯粹显示时间。不支持自定义字体或颜色。

#### 3.5 控制栏

- **可用性**：优秀。顶部时钟 + 场景按钮 + 功能按钮，自动 3s 隐藏（Vue 版）/ 点击切换显隐。支持键盘快捷键（方向键切换场景，空格/ESC 显隐控制栏，d 调暗，f 聚焦，m 静音）。
- **体验**：blur 毛玻璃底部栏，hover/active 反馈，过渡动画流畅。
- **完整性**：所有核心功能入口都在控制栏上可触及（场景切换、静音、调暗、聚焦、计时器、声音面板）。

#### 3.6 粒子系统

- **可用性**：中等。Emitter+Lifecycle 架构已跑通（campfire 场景），Perlin 噪声湍流算法实现。但 currentParticles 与 Emitter 内部分别管理数据，存在双源同步问题。
- **体验**：粒子为完美正圆形（ShaderMaterial discard 裁剪），尺寸/颜色/Alpha 生命周期正确。目前效果离需求规格仍有差距（火焰锥形不够自然、木柴层无 3D 立体感）。
- **完整性**：仅 campfire 有完整发射器配置（火焰/火星/尘埃 3 个发射源）。ocean / rainy-window / forest 场景配置存在但粒子生成函数返回空数组。其他 3 个场景（rain / snow / cafe）尚无粒子定义。

#### 3.7 PWA 支持

- **可用性**：Service Worker 已注册，manifest.json 配置完整，可离线缓存静态资源。
- **体验**：安装弹窗（beforeinstallprompt）未实现；已安装 PWA 启动画面使用默认 Android 启动屏。
- **完整性**：Workbox / 自定义 SW cache 策略未配置，视频/音频流不走缓存。

## 4. 问题清单

### P0 — 功能不可用，阻塞核心流程

| # | 问题 | 模块 | 描述 |
|---|------|------|------|
| 1 | 粒子场景仅 campfire 可用 | 场景渲染 | ocean / rainy-window / forest 配置存在但 `generateParticles()` 返回空数组，Three.js 场景无粒子渲染 |
| 2 | 粒子引擎双源同步问题 | 粒子系统 | `useParticleEngine` 中 emitters 和 currentParticles 各自维护数据，每帧都做全量 Buffer copy，冗余且可能不一致 |
| 3 | 番茄钟完成无统计保留 | 计时 | `timerStats` 容器永远为空；页面刷新后番茄计数丢失 |

### P1 — 功能可用但体验差

| # | 问题 | 模块 | 描述 |
|---|------|------|------|
| 4 | 场景视频首屏加载慢 | 场景渲染 | 视频源托管在 COS，首次访问需 DNS + 预签名 URL 签名 + 缓冲区填充，耗时 1-3s |
| 5 | 声音面板操作路径深 | 声音混音 | 调音量：点击🎵按钮 → 面板滑出 → 找到音效 → 拖滑块，共 4 步 |
| 6 | 倒计时设置不生效 | 计时 | TimerPanel 中 `input[type=number]` 没有 v-model 绑定到 `timer.duration`，输入值无效 |
| 7 | 聚焦时钟仅在 10s 间隔更新 | 聚焦模式 | `setInterval` 间隔 10 秒，切换场景时用户可能看到过时时间 |
| 8 | 粒子调试元素残留 | 粒子系统 | white-noise 版页面左上角始终显示黄色调试文字（`#particle-debug`），橙色球体仍在场景中 |

### P2 — 锦上添花 / 技术债务

| # | 问题 | 模块 | 描述 |
|---|------|------|------|
| 9 | 缺乏场景预览缩略图 | 场景渲染 | 场景切换按钮仅有 emoji icon，用户无法预先看到场景画面 |
| 10 | 无安装 PWA 弹窗 | PWA | 未监听 `beforeinstallprompt` 事件，用户无法快速安装到桌面 |
| 11 | 视频缓存策略缺失 | 性能 | 视频流每次打开重新请求 COS 预签名 URL，未使用 Cache API 或 IndexedDB 缓存 |
| 12 | 代码库版本碎片化 | 架构 | 同一产品存在 3 个独立前端实现：React Native (Expo)、Vue 3 + TS、Vanilla JS PWA，功能不一致 |
| 13 | 无用户系统 | 基础架构 | 所有配置存在 localStorage，多设备无法同步，无云端备份 |
| 14 | 缺少国际化 | 基础架构 | 所有 UI 文本硬编码为中文 |
| 15 | 白噪音缺少"随机组合"功能 | 声音混音 | 用户无法一键"随机给我一组音效"——当前需要手动点选 |

## 7. 下一步建议

### 立即整改（P0 + 快速赢取）

1. **修复倒计时输入绑定** — TimerPanel 中 `v-model` 和 `watch` 缺失，导致用户输入的分/秒值未传递到 `timer.duration`
2. **清理粒子调试元素** — 移除 `#particle-debug` 覆盖层和场景中的橙色 debug 球体
3. **实现番茄钟统计** — 在 localStorage 中持久化番茄完成次数和今日总专注时长，渲染到 `timerStats` 容器
4. **增加快捷音量控制** — CtrlBar 上增加总音量滑动条（或拖拽手势控制当前场景主音量），减少操作路径

### 迭代规划（下一版本）

5. **粒子系统全场景覆盖** — 将 campfire 的 Emitter+Lifecycle 架构推广到 ocean / rainy-window / forest 场景，移除旧的 `generateParticles` 返回空数组分支
6. **视频预加载优化** — 预先加载相邻 2 个场景的视频到 `HTMLVideoElement` 缓冲区，切换时无白屏
7. **达成粒子效果达标** — 按 REQUIREMENTS.md 中 Phase 2/3 计划：火星发射器、木柴层 3D 立体化、尘埃层丁达尔效应、不透明度滑块
8. **统一代码架构** — 将 Vanilla JS PWA（web/）和 Expo React Native（StudyCompanion/）的功能合并入 white-noise Vue 3 主版本，减少维护负担

### 长期方向

9. **用户系统 + 云端同步** — 简单邮箱/微信登录，混音预设、番茄统计跨设备同步
