# 白噪音篝火页 · 粒子系统重构需求文档

## 1. 产品全景

| 维度 | 说明 |
|------|------|
| **产品名** | WhiteNoise · 陪伴学习 |
| **定位** | 提供沉浸式白噪音 + 视觉效果的学习辅助应用 |
| **目标用户** | 需要专注/放松/入睡的学生、远程工作者 |
| **核心场景** | 打开网页 → 选择场景（篝火/雨声/海浪等） → 自动播放音画 → 调节音量 → 沉浸使用 |
| **当前阶段** | ParticleCanvas 替代原 VideoCanvas，Three.js 粒子渲染已跑通但效果不达标 |
| **技术栈** | Vue 3 + TypeScript + Three.js + Vite + Pinia |

### 当前粒子系统问题清单（P0/P1/P2）

| 优先级 | 问题 | 描述 |
|--------|------|------|
| **P0** | 火焰形态失真 | 粒子无火舌结构，呈现扁平光带或爆炸喷射，无真实篝火的锥形蓬松形态 |
| **P0** | 动态节奏错误 | 粒子运动速度过快（爆炸式），与白噪音 3-5 秒舒缓周期不匹配 |
| **P1** | 颜色分层缺失 | 缺少焰心(蓝白)→中层(金黄)→外焰(橙红)三层渐变的完整映射 |
| **P1** | 木柴基底缺失 | 无 3 段交错堆叠木柴，火焰悬浮无真实感 |
| **P1** | 方块像素化 | 粒子边缘无 `discard` 裁剪，GL_POINT 渲染默认方形 |
| **P2** | 尘埃层不融合 | 环境微尘与火焰缺乏光学交互感（丁达尔效应） |
| **P2** | 缺少发射器架构 | 当前粒子位置固定+正弦动画，无 emitter+lifecycle 体系 |

## 2. 竞品技术方案对比

| 维度 | 当前方案 | 行业最佳实践（firesimulation） | 行业最佳实践（three-nebula） | 差距 |
|------|----------|------|------|------|
| **粒子架构** | 固定位置+正弦动画 | 发射器 + 粒子生命周期 | 多发射器 + 行为链 + JSON 配置 | ← 需全面重构 |
| **火焰形状** | 散点均匀分布 | 元胞自动机温度扩散 + 锥形约束 | 锥形发射 + 湍流扰动 + 重力 | ← 需锥形发射区域 |
| **颜色映射** | 硬编码 RGB 三段 | 温度→颜色 LUT（白→黄→橙→红→暗） | 生命周期色渐变 + 随机偏移 | ← 需温度 LUT |
| **粒子生命周期** | 无（无限存活） | 像素有燃烧/冷却周期 | 发射→上升→膨胀→收缩→消散 | ← 核心缺失 |
| **动态控制** | Math.sin 单一频率 | Perlin 噪声湍流 | 多频湍流 + 风力 + 重力扰动 | ← 需噪声引擎 |
| **性能策略** | 单层全量更新 | Canvas 2D 像素操作 | GPU 粒子 + LOD + 对象池 | ← 中等可接受 |
| **尘埃叠加** | 同场景混入 | 独立 Canvas 层 | 独立发射源 + 滤色混合 | ← 需独立发射源 |

## 3. 技术方案: 基于 Emitter+Lifecycle 的粒子系统

### 3.1 核心架构（参考 three.proton / three-nebula 模式）

```
ParticleSystem
  └── Emitter[]
       ├── EmissionShape (Point / Cone / Box / Sphere)
       ├── SpawnRate (particles/sec)
       ├── MaxParticles
       └── Behaviour[]
            ├── ColorOverLife: startColor → endColor
            ├── SizeOverLife: startSize → peakSize → endSize
            ├── AlphaOverLife: fadeIn → hold → fadeOut
            ├── ForceOverLife: gravity, wind, turbulence
            └── VelocityOverLife: initial speed + direction + randomness
```

### 3.2 粒子数据结构（重构 `ParticleData`）

```typescript
interface Particle {
  // 实时状态
  x, y, z: number
  vx, vy, vz: number     // 速度（新增！核心缺失）
  size: number
  alpha: number
  r, g, b: number

  // 生命周期
  age: number              // 当前存活时间
  lifespan: number         // 总寿命（到期重生成）

  // 出生状态（发射时确定）
  originX, originY, originZ: number
  startSize: number
  peakSize: number
  endSize: number
  startColor, endColor: [number, number, number]
  phaseOffset: number
}
```

### 3.3 发射器配置（对应三个独立发射源）

#### 发射源 A：火焰主发射器（篝火火舌）

```
发射形状: 锥形（底部半径 1.2，顶部半径 0.3，高度 3.0）
发射率:  200 particles/sec
最大粒子: 800

颜色生命周期:
  age 0~20%:   (255, 220, 180) 淡蓝白 → 焰心
  age 20~50%:  (255, 200, 80)  暖金黄 → 主亮部
  age 50~80%:  (255, 140, 50)  橙黄 → 过渡
  age 80~100%: (200, 80, 30)   橙红半透明 → 消散

尺寸生命周期:
  start: 0.5 → peak: 3.0 (40% 寿命时) → end: 1.0

速度:
  vy: 0.8~2.0 (向上)
  vx: ±0.3 (随机偏移)
  vz: ±0.2 (随机偏移)
  受湍流扰动

湍流: Perlin 噪声，频率 0.3，振幅 0.5，周期 3-5 秒
```

#### 发射源 B：火星发射器（极少量点缀）

```
发射形状: 点（从木柴缝隙发射）
发射率:  5-8 particles/sec（极少！）
最大粒子: 30

颜色: (255, 220, 120) 亮金色 → (255, 160, 40) 暗淡消散
尺寸: start 0.3 → peak 1.0 → end 0.1
速度: vy 0.3~0.8（慢速上浮），vx/vz ±0.2

生命周期: 4-6 秒
受重力: vy 逐渐衰减至 0
```

#### 发射源 C：尘埃环境发射器（丁达尔效应 + 滤色叠加）

```
发射形状: 盒形（范围 8 x 5 x 6）
发射率:  50 particles/sec
最大粒子: 300

颜色: (255, 180, 100) 暖橙极淡
尺寸: 0.3~0.8
速度: vy 0.05~0.15（几乎静止漂浮）
透明度: 0.01~0.04（极淡）

混合模式: AdditiveBlending（滤色）
无湍流，仅极慢布朗运动
```

### 3.4 木柴层（静态独立渲染）

```
3 段交错堆叠木柴
每段用 ~500 粒子组成椭圆体分布
颜色: 正面暖橙 / 背面深棕褐+焦黑
不写入深度缓冲，永远在火焰层下方
```
