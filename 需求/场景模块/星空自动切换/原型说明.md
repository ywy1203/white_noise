# 星空自动切换 — 原型说明

## 页面清单

仅修改**底部控制栏**，不新增页面。

## 框线示意

### 改造前底部栏
```
┌──────────────────────────────────────────────────────────┐
│  ←│ 🔥壁炉 │→      🕐14:30    ▶  ⏹  [25min]           │
└──────────────────────────────────────────────────────────┘
  场景切换              计时器   播放 重置  倒计时预设
```

### 改造后底部栏
```
┌──────────────────────────────────────────────────────────────┐
│  ←│ 🔥壁炉 │→      🕐14:30    ▶  ⏹  [25min]  🔁 ← 新增     │
└──────────────────────────────────────────────────────────────┘
  场景切换              计时器   播放 重置  倒计时预设  自动轮播
```

### 🔁 长按弹出面板
```
┌──────────────┐
│ 自动切换间隔   │
├──────────────┤
│ ○ 15 分钟    │
│ ● 30 分钟    │ ← 默认
│ ○ 45 分钟    │
│ ○ 60 分钟    │
└──────────────┘
```

## 交互流程

### 流程一：开启自动轮播
点击 🔁 → 按钮高亮（蓝色）→ 启动定时器 → 每 30 分钟调用切换场景

### 流程二：设置间隔
长按 🔁 → 弹出设置面板 → 选择间隔 → 面板关闭 → 定时器按新间隔运行

### 流程三：手动切换重置
用户左/右滑切换场景 → 定时器重置，重新计时

### 流程四：倒计时结束
倒计时归零 → 自动轮播停止 → 🔁 恢复未选中状态

## 页面跳转关系

无新增页面。所有交互在当前底部栏 + 弹出面板完成。

## 关键字段/状态

| 元素 | 状态 | 说明 |
|------|------|------|
| 🔁 按钮 | 关闭（灰色）| 默认状态 |
| 🔁 按钮 | 开启（蓝色）| 自动轮播运行中 |
| 🔁 按钮 | 置灰不可点 | 仅一个场景时 |
| 间隔面板 | 弹出/关闭 | 长按开关切换 |
